Front-end é tudo aquilo
que o usuário vê na web

Três tecnologias permitem criar sites mais interessantes.
Abaixo resumimos cada uma delas.

HTML

Linguagem de marcação. Fornece a estrutura da página definindo os elementos.

CSS

Linguagem de estilo. Define a aparência visual e o layout dos elementos definidos em HTML.

JavaScript

Linguagem de programação. Adiciona interação com o usuário e atualização no conteúdo HTML e CSS.
